# SDP-Analysis-Dashboard

このプロジェクトは、SDP（Software Defined Perimeter）分析ダッシュボードのためのPythonアプリケーションです。

## 構成ファイル
- `app.py` : メインアプリケーションファイル
- `requirements.txt` : 必要なPythonライブラリ一覧

## セットアップ方法
1. 必要なライブラリのインストール

```bash
pip install -r requirements.txt
```

2. アプリケーションの実行

```bash
python app.py
```
